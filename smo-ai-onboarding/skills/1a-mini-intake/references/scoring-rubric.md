# Scoring Rubric (Sub-step 1.A)

5-dim, threshold ≥9.0, MENA-fit floor 8.0.

## Specificity
10: All fields concrete. <5: Critical fields missing.

## Evidence
10: URL verified, names match. <5: URL fake/typo'd.

## Differentiation
10: Persona unambiguous, priority strict. <5: defaults accepted.

## Actionability
10: Output enables 1b without clarification. <5: blocks 1b.

## MENA-fit (FLOOR 8.0)
10: Language + dialect + per-channel set. <8: language defaulted without asking.

## Composite
```
composite = avg(5 dims)
threshold: ≥9.0 AND every dim ≥7
```

## Gap-Fill Hints

- Specificity: ask for concrete legal name, not generic.
- Evidence: ask for live URL.
- Differentiation: force pick closest persona.
- Actionability: force priority pick (which ONE first?).
- MENA-fit: ask language explicitly.

Max 3 retries before founder escalation.
