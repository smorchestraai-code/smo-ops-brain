# Source Map (anti-hallucination spine)

Every brain field maps to >=1 Phase 1 source. No source = INPUT-GAP, never invent.

## Format

```
| Brain file | Field | Phase 1 source(s) | Status |
|---|---|---|---|
| About-Me/personal-background | origin story | interview Q4, memory operating-preferences | OK |
| Project/icp | buyer pain | interview Q12, customer-conversations extract | OK |
| Project/pricing-strategy | enterprise tier | (none) | INPUT-GAP |
```

## INPUT-GAP Handling

- Mark the field INPUT-GAP
- Builder writes placeholder: "[INPUT-GAP: not captured in Phase 1. Surface to founder.]"
- Validator (2f) lists all INPUT-GAPs for operator review
- Do NOT fabricate to fill the gap
